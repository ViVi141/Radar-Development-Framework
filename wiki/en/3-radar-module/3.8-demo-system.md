# Radar Demo System

## RDF_RadarAutoRunner

Singleton demo driver. Manages scan loop, callbacks, and world markers.

| Method | Description |
|--------|-------------|
| `static GetInstance()` | Get singleton (lazy init) |
| `static void StartWithConfig(RDF_RadarDemoConfig cfg)` | Start with preset |
| `static void Stop()` | Stop and cleanup |
| `static void SetScanCompleteHandler(RDF_LidarScanCompleteHandler handler)` | Register callback |
| `static GetLastSamples()` | Get last scan copy (defensive) |
| `static bool IsRunning()` | Whether demo is running |

---

## RDF_RadarAutoBootstrap

`modded class SCR_BaseGameMode`. Auto-starts radar demo at game start. **Disabled by default**.

```c
SCR_BaseGameMode.SetRadarBootstrapEnabled(true);
```

---
