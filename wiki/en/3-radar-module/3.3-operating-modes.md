# Radar Operating Modes

All mode classes extend `RDF_RadarMode`. `RDF_RadarScanner` instantiates them based on `ERadarMode`.

---

## Modes

| Mode | ERadarMode | Description |
|------|------------|-------------|
| **Pulse** | PULSE | Distance quantization (c·τ/2); max unambiguous range c/(2·PRF) |
| **CW** | CW | Continuous wave; velocity only; distance set to 0 |
| **FMCW** | FMCW | Linear chirp; high resolution distance (c/(2·B)) |
| **Phased Array** | PHASED_ARRAY | Multi-beam SNR penalty: ΔL = 10·log(beamCount) |

---

## Mode Selection

Set via `RDF_RadarSettings.m_RadarMode` or `RDF_RadarScanner.SetRadarMode(ERadarMode mode)`.
