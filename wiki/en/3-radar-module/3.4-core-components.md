# Radar Core Components

## RDF_EMWaveParameters

Electromagnetic wave parameters: frequency, wavelength, power, PRF, antenna gain, etc.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `m_CarrierFrequency` | float | 10e9 | Carrier frequency (Hz) |
| `m_Wavelength` | float | 0.03 | Wavelength (m) |
| `m_BandName` | string | "X" | Band (L/S/C/X/Ku/K/Ka/V/W) |
| `m_TransmitPower` | float | 1000.0 | Transmit power (W) |
| `m_PulseWidth` | float | 1e-6 | Pulse width (s) |
| `m_PRF` | float | 1000.0 | Pulse repetition frequency (Hz) |
| `m_AntennaGain` | float | 30.0 | Antenna gain (dBi) |
| `m_BeamwidthAzimuth` | float | 3.0 | Azimuth beamwidth (deg) |
| `m_BeamwidthElevation` | float | 3.0 | Elevation beamwidth (deg) |

**Methods**: `CalculateWavelength()`, `DetermineBand()`, `Validate()`, `GetDescription()`

---

## RDF_RadarSample

Extends `RDF_LidarSample`. Adds: `m_ReceivedPower`, `m_SignalToNoiseRatio`, `m_RadarCrossSection`, `m_PathLoss`, `m_AtmosphericLoss`, `m_DopplerFrequency`, `m_TargetVelocity`, `m_TimeOfFlight`, etc.

---

## RDF_RadarSettings

Extends `RDF_LidarSettings`. Adds: `m_EMWaveParams`, `m_RadarMode`, `m_DetectionThreshold`, `m_EnableDopplerProcessing`, `m_EnableMTI`, `m_EnableMTD`, atmospheric/rain/RCS/CFAR options, etc.

---
