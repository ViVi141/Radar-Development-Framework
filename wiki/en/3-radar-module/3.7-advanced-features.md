# Radar Advanced Features

## SAR Processor (RDF_SARProcessor)

Synthetic Aperture Radar processor for imaging.

```c
void AddScan(array<ref RDF_LidarSample> samples, vector platformPos)
void GenerateSARImage(float areaSize, int resolution, vector areaCenter, ...)
void VisualiseImage(...)
void Reset()
int GetSnapshotCount()
```

`m_MaxSnapshots` (default 32) limits history snapshots.

---

## ECM & Jamming (RDF_JammingModel)

| Method | Description |
|--------|-------------|
| `CalculateNoiseJammingJS(Pj, Gj, Gr, λ, Rj, Br)` | Noise jamming J/S (dB) |
| `ApplyNoiseJamming(samples, jsDB, thresholdDB)` | Apply noise jamming to samples |
| `InjectFalseTargets(samples, count, x, y, z, ...)` | Inject deceptive false targets |
| `CalculateChaffRCS(numDipoles, wavelength)` | Chaff cloud RCS (m²) |

## Target Classification (RDF_TargetClassifier)

**ERadarTargetClass**: UNKNOWN, INFANTRY, VEHICLE_LIGHT, VEHICLE_HEAVY, AIRCRAFT, STATIC_OBJECT, SMALL_UAV, NAVAL

| Method | Description |
|--------|-------------|
| `ClassifyTarget(sample)` | Classify single sample by RCS + velocity rules |
| `ClassifyAll(samples, out outClasses)` | Batch classify |
| `GetClassName(cls)` | Enum to readable string |
| `PrintStats(samples)` | Print classification stats to log |
