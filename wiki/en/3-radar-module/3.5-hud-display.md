# Radar HUD & Display

## RDF_RadarHUD

Fully script-driven radar HUD. Built with `WorkspaceWidget.CreateWidgetInWorkspace`; no `.layout` file required. Position: bottom-left (X=20, Y=820 at 1920×1080).

---

## Layout

```
+==========================================+
| [>] RADAR              X-Band Pulse     |  <- Title + mode
+------------------------------------------+
|               N (North)                  |
|         CanvasWidget PPI circle          |
|   W (West)   (+) Player   E (East)       |
|               S (South)                  |
|  Inner ring = 50% range, Outer = 100%    |
+------------------------------------------+
| SNR  65.2 dB   Hits  23 / 512 rays      |
| RCS  +18.0 dBsm   Vel  3.2 m/s          |
| Range  30 m  ..  1447 m                  |
+------------------------------------------+
```

---

## Target Color Coding

| Condition | Color (ARGB) | Radius |
|-----------|--------------|--------|
| Entity, RCS > 10 dBsm | Bright green #00FF44 | 4 px |
| Entity, RCS ≤ 10 dBsm | Cyan #00CCCC | 3 px |
| Terrain, SNR > 20 dB | Yellow #FFCC00 | 2 px |
| Terrain (weak) | Dark orange #AA6600 | 2 px |

---

## Key Methods

```c
static RDF_RadarHUD GetInstance()
static void Show()
static void Hide()
static bool IsVisible()
static void SetMode(string modeName)
static void SetDisplayRange(float rangeM)
static void FeedSamples(array<ref RDF_LidarSample> samples)  // Without AutoRunner
static void AttachToAutoRunner()
static void DetachFromAutoRunner()
```

**Throttle**: HUD updates at most every 0.5 s to prevent flicker.
