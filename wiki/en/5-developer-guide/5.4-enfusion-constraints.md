# Enfusion Script Constraints

| Constraint | Correct Approach |
|------------|------------------|
| No ternary operator | Use `if/else` instead of `? :` |
| No functional cast | Use `(float)x` instead of `float(x)` |
| No `string.ToLower()` | Compare against upper/lower literals directly |
| No enum integer conversion | Pre-fill `EMyEnum[]` array for lookup |
| `string.Format` uses position specifiers | Use `%1 %2` instead of `%d %.1f` |
| `.ToString()` not on temp expressions | Assign to local variable first |
| `out` is a keyword | Use different names, e.g. `outSamples` |
| Non-ASCII causes syntax errors | Keep files ASCII-only |
