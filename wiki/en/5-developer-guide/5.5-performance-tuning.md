# Performance Tuning

## Radar

- Lower `m_RayCount` (default 512) or increase `m_MinTickInterval` to reduce scan frequency
- Disable atmospheric/rain models if not needed
- Disable CFAR if not required

---

## LiDAR

- Reduce `m_RayCount` or `m_RaySegments` for cheaper rendering
- Enable `m_UseBatchedMesh` for high ray counts (fewer Shape calls)
- Use logic-only scan (no Visualizer) when no display is needed

---

## HUD

- HUD refresh is throttled (`UPDATE_INTERVAL = 0.5 s`) to prevent flicker
- Avoid excessive `SetDrawCommands` calls; cache static draw commands

---

## Network (LiDAR)

- Use RLE compression for large payloads
- Server-side: stream/partition CSV to reduce peak memory
