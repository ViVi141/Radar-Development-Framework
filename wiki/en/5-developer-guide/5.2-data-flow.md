# Data Flow

## LiDAR Data Flow

```
RDF_LidarSettings
       |
       v
RDF_LidarScanner.Scan(entity)
  +-- SampleStrategy.BuildDirection() x N
  +-- Trace(ray) -> RDF_LidarSample
  +-- Returns array<ref RDF_LidarSample>
       |
       v
RDF_LidarVisualizer.RenderWithSamples()   (optional)
       |
       v
RDF_LidarScanCompleteHandler.OnScanComplete()
```

---

## Radar Data Flow

```
RDF_RadarSettings + RDF_EMWaveParameters
       |
       v
RDF_RadarScanner.Scan(entity)
  +-- Inherits Lidar ray Trace flow
  +-- ApplyRadarPhysics(sample) for each hit:
       1. FSPL
       2. Atmospheric attenuation
       3. Rain attenuation
       4. RCS estimation
       5. Radar equation
       6. Range gate
       7. Noise -> SNR
       8. Doppler
       9. Detection threshold
      10. Clutter filter
       |
       v
RDF_RadarWorldMarkerDisplay / RDF_RadarTextDisplay / RDF_RadarHUD
```
