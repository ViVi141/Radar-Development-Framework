# RDF_LidarSettings

Scanner configuration for `RDF_LidarScanner`. Values are clamped to safe ranges at runtime via `Validate()`.

---

## Fields

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `m_Enabled` | bool | true | Enable scanning |
| `m_Range` | float | 50.0 | Scan radius (m). Clamped to [0.1, 100000.0] |
| `m_RayCount` | int | 512 | Number of rays. Minimum 1 at runtime |
| `m_UpdateInterval` | float | 5.0 | Scan interval (s). Minimum 0.01 |
| `m_OriginOffset` | vector | "0 0 0" | Origin offset; used with m_UseLocalOffset |
| `m_TraceFlags` | int | TraceFlags.WORLD \| TraceFlags.ENTS | Trace flags |
| `m_LayerMask` | int | EPhysicsLayerPresets.Projectile | Physics layer mask |
| `m_UseBoundsCenter` | bool | true | Use entity bounding-box center as origin |
| `m_UseLocalOffset` | bool | true | Treat m_OriginOffset as local-space offset |

---

## Methods

| Method | Description |
|--------|-------------|
| `void Validate()` | Performs defensive validation and clamping. Called automatically before `Scan()`. |
