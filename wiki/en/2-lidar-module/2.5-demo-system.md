# LiDAR Demo System

## RDF_LidarAutoRunner

Single entry point for LiDAR demos. Controlled via `SetDemoEnabled` and `SetDemoConfig` / `StartWithConfig`.

---

## Key Methods

| Method | Description |
|--------|-------------|
| `static void SetDemoEnabled(bool enabled)` | Enable / disable demo |
| `static bool IsDemoEnabled()` | Returns demo state |
| `static void StartWithConfig(RDF_LidarDemoConfig cfg)` | Apply config and start demo |
| `static void SetDemoConfig(RDF_LidarDemoConfig cfg)` | Set config (does not auto-start) |
| `static RDF_LidarDemoConfig GetDemoConfig()` | Get current config |
| `static void SetScanCompleteHandler(RDF_LidarScanCompleteHandler handler)` | Register scan-complete callback |
| `static void SetDemoSampleStrategy(RDF_LidarSampleStrategy strategy)` | Set sampling strategy |
| `static void SetDemoColorStrategy(RDF_LidarColorStrategy strategy)` | Set color strategy |
| `static void SetDemoRenderWorld(bool renderWorld)` | Point-cloud-only vs world+point-cloud |

---

## RDF_LidarDemoConfig

Configuration object. Create via factory presets:

- `CreateDefault(int rayCount = 256)`
- `CreateThreeColor(int rayCount = 512)`
- `CreateHemisphere(int rayCount = 256)`
- `CreateConical(float halfAngleDeg, int rayCount)`
- `CreateSweep(float halfAngleDeg, float sweepWidthDeg, float sweepSpeedDegPerSec, int rayCount)`

---

## RDF_LidarDemoCycler

Cycles through sampling strategies:

```c
static void Cycle(int rayCount = 256)
static void StartIndex(int index, int rayCount = 256)
static void StartAutoCycle(float interval = 10.0)
static void StopAutoCycle()
static bool IsAutoCycling()
```
