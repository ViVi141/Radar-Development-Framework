# Sample Strategies

Sampling strategies define how ray directions are generated. Implement `RDF_LidarSampleStrategy::BuildDirection(int index, int count)` and inject via `SetSampleStrategy()`.

---

## Built-in Strategies

| Strategy | Description |
|----------|-------------|
| **RDF_UniformSampleStrategy** | Uniform sphere (Fibonacci-like) |
| **RDF_HemisphereSampleStrategy** | Hemisphere (e.g. upward) |
| **RDF_ConicalSampleStrategy** | Conical sector |
| **RDF_StratifiedSampleStrategy** | Stratified sampling |
| **RDF_ScanlineSampleStrategy** | Scanline pattern |
| **RDF_SweepSampleStrategy** | Rotating sector (radar-style) |

---

## RDF_SweepSampleStrategy

Radar-style rotating sector.

```c
RDF_SweepSampleStrategy(float halfAngleDeg = 30.0, float sweepWidthDeg = 20.0, float sweepSpeedDegPerSec = 45.0)
```

- **halfAngleDeg** — Cone half-angle (degrees)
- **sweepWidthDeg** — Sector width (degrees)
- **sweepSpeedDegPerSec** — Rotation speed (deg/s)

Ray directions are distributed across the sector and the sector rotates with world time.
