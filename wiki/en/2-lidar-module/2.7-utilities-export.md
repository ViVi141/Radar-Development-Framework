# LiDAR Utilities & Export

## RDF_LidarSubjectResolver

Resolves scan subject (local player or vehicle).

| Method | Description |
|--------|-------------|
| `static IEntity ResolveLocalSubject(bool preferVehicle = true)` | Resolve local player; in vehicle returns vehicle root if preferVehicle |
| `static IEntity ResolveSubject(IEntity player, bool preferVehicle = true)` | Resolve subject for given player |
| `static vector ResolveOrigin(IEntity player = null, bool preferVehicle = true)` | Resolve subject origin |

---

## RDF_LidarExport

Export scan results to CSV.

| Method | Description |
|--------|-------------|
| `static string GetCSVHeader()` | CSV header row |
| `static string SampleToCSVRow(RDF_LidarSample sample)` | Format single sample |
| `static void PrintCSVToConsole(array<ref RDF_LidarSample> samples)` | Print full scan to console |
| `static bool ExportToFile(array<ref RDF_LidarSample> samples, string path)` | Write to file (overwrite) |
| `static bool AppendToFile(array<ref RDF_LidarSample> samples, string path, bool writeHeaderIfNew = true)` | Append to file |

---

## RDF_LidarSampleUtils

Statistics and filtering.

| Method | Description |
|--------|-------------|
| `static RDF_LidarSample GetClosestHit(array<ref RDF_LidarSample> samples)` | Closest hit |
| `static RDF_LidarSample GetFurthestHit(array<ref RDF_LidarSample> samples)` | Furthest hit |
| `static int GetHitCount(array<ref RDF_LidarSample> samples)` | Hit count |
| `static void GetHitsInRange(samples, minDist, maxDist, outSamples)` | Filter by range |
| `static float GetAverageDistance(samples, bool hitsOnly = true)` | Average distance |

---

## RDF_LidarScanCompleteHandler

Callback base class. Override `OnScanComplete(array<ref RDF_LidarSample> samples)` to process results.
