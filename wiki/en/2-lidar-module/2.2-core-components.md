# LiDAR Core Components

## RDF_LidarSettings

Scanner configuration. Clamped at runtime to safe ranges.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `m_Enabled` | bool | true | Enable scanning |
| `m_Range` | float | 50.0 | Scan radius (m). Clamped to [0.1, 100000.0] |
| `m_RayCount` | int | 512 | Number of rays. Minimum 1 |
| `m_UpdateInterval` | float | 5.0 | Scan interval (s). Minimum 0.01 |
| `m_OriginOffset` | vector | "0 0 0" | Origin offset (with m_UseLocalOffset) |
| `m_TraceFlags` | int | WORLD \| ENTS | Trace flags |
| `m_LayerMask` | int | Projectile | Physics layer mask |
| `m_UseBoundsCenter` | bool | true | Use bounding-box center as origin |
| `m_UseLocalOffset` | bool | true | m_OriginOffset in local space |

**Methods**:
- `void Validate()` — Validation and clamping (called automatically before scan).

---

## RDF_LidarSample

Single-ray sample structure.

| Field | Type | Description |
|-------|------|-------------|
| `m_Index` | int | Ray index |
| `m_Hit` | bool | Whether hit |
| `m_Start` | vector | Ray start |
| `m_End` | vector | Ray end (max distance) |
| `m_Dir` | vector | Unit direction |
| `m_HitPos` | vector | Hit position (or ray end) |
| `m_Distance` | float | Hit distance (or max range) |
| `m_Entity` | IEntity | Hit entity (if any) |
| `m_ColliderName` | string | Collider name |
| `m_Surface` | GameMaterial | Hit surface material |

---

## RDF_LidarScanner

Main scanner class.

```c
RDF_LidarScanner(RDF_LidarSettings settings = null)
RDF_LidarSettings GetSettings()
void Scan(IEntity subject, array<ref RDF_LidarSample> outSamples)
void SetSampleStrategy(RDF_LidarSampleStrategy strategy)
RDF_LidarSampleStrategy GetSampleStrategy()
```

**Default sampling strategy**: `RDF_UniformSampleStrategy`.
