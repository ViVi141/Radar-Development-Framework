# RDF_LidarSample

Single-ray sample result structure. Contains geometry and hit information.

---

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `m_Index` | int | Ray index |
| `m_Hit` | bool | Whether the ray hit something |
| `m_Start` | vector | Ray start position |
| `m_End` | vector | Ray end (max distance if no hit) |
| `m_Dir` | vector | Unit direction vector |
| `m_HitPos` | vector | Hit position (or ray end if no hit) |
| `m_Distance` | float | Hit distance (or max range if no hit) |
| `m_Entity` | IEntity | Hit entity (if any) |
| `m_ColliderName` | string | Name of hit collider |
| `m_Surface` | GameMaterial | Hit surface material (if available) |

---

## Inheritance

`RDF_RadarSample` extends `RDF_LidarSample` and adds radar-specific fields (SNR, RCS, Doppler, etc.). When using `RDF_RadarScanner.Scan()`, cast samples to `RDF_RadarSample` to access radar data.
