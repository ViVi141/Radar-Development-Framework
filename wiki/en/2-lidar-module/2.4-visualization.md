# LiDAR Visualization

## RDF_LidarVisualizer

Renders point cloud and optional rays. Uses `RDF_LidarVisualSettings` for parameters.

```c
RDF_LidarVisualizer(RDF_LidarVisualSettings settings = null)
void Render(IEntity subject, RDF_LidarScanner scanner)
ref array<ref RDF_LidarSample> GetLastSamples()  // Returns defensive copy
```

---

## RDF_LidarVisualSettings

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `m_DrawPoints` | bool | true | Draw hit points |
| `m_DrawRays` | bool | true | Draw rays |
| `m_ShowHitsOnly` | bool | false | Draw only hits |
| `m_PointSize` | float | 0.08 | Point sphere radius |
| `m_RayAlpha` | float | 0.25 | Ray alpha |
| `m_RaySegments` | int | 6 | Ray segments for gradient |
| `m_UseDistanceGradient` | bool | true | Distance-based gradient coloring |
| `m_RenderWorld` | bool | true | true = game world + point cloud; false = point-cloud only |
| `m_UseBatchedMesh` | bool | false | Batch mesh rendering (fewer Shape calls) |

---

## Color Strategies

Implement `RDF_LidarColorStrategy` and inject into visualizer:

- `int BuildPointColor(float dist, bool hit, float lastRange, RDF_LidarVisualSettings settings)`
- `int BuildRayColorAtT(float t, bool hit, RDF_LidarVisualSettings settings)`

**Built-in implementations**:
- `RDF_DefaultColorStrategy` — Distance gradient
- `RDF_IndexColorStrategy` — Index-based coloring
- `RDF_ThreeColorStrategy` — Near/mid/far (green→yellow→red)
