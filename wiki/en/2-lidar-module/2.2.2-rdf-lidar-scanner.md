# RDF_LidarScanner

Main LiDAR scanner class. Performs ray tracing and fills an array of `RDF_LidarSample`.

---

## Constructor

```c
RDF_LidarScanner(RDF_LidarSettings settings = null)
```

Creates a scanner with default or custom settings.

---

## Methods

| Method | Description |
|--------|-------------|
| `RDF_LidarSettings GetSettings()` | Returns the settings reference |
| `void Scan(IEntity subject, array<ref RDF_LidarSample> outSamples)` | Performs scan; fills outSamples. Calls `Validate()` at entry |
| `void SetSampleStrategy(RDF_LidarSampleStrategy strategy)` | Injects a custom sampling strategy |
| `RDF_LidarSampleStrategy GetSampleStrategy()` | Returns the current strategy |

---

## Usage Example

```c
RDF_LidarScanner scanner = new RDF_LidarScanner();
scanner.GetSettings().m_Range = 100.0;
scanner.GetSettings().m_RayCount = 256;

array<ref RDF_LidarSample> samples = new array<ref RDF_LidarSample>();
IEntity subject = RDF_LidarSubjectResolver.ResolveLocalSubject(true);
scanner.Scan(subject, samples);
```

**Default strategy**: `RDF_UniformSampleStrategy`.
