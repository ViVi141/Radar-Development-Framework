# Radar API Reference

Summary of the main public Radar classes. For full details see `docs/RADAR_API.md` in the repository.

---

## Core

| Class | Description |
|-------|-------------|
| **RDF_EMWaveParameters** | m_CarrierFrequency, m_TransmitPower, m_AntennaGain, m_BandName |
| **RDF_RadarSample** | Extends RDF_LidarSample; adds m_SignalToNoiseRatio, m_RadarCrossSection, m_DopplerFrequency |
| **RDF_RadarSettings** | m_EMWaveParams, m_RadarMode, m_DetectionThreshold, m_EnableDopplerProcessing |
| **RDF_RadarScanner** | Scan(), GetRadarSettings(), SetRadarMode() |

---

## Physics

| Class | Description |
|-------|-------------|
| **RDF_RadarPropagation** | CalculateFSPL, CalculateAtmosphericAttenuation, CalculateRainAttenuation |
| **RDF_RCSModel** | EstimateEntityRCS, GetMaterialReflectivity |
| **RDF_RadarEquation** | CalculateReceivedPower, CalculateSNR |
| **RDF_DopplerProcessor** | CalculateDopplerShift, CalculateRadialVelocity |
| **RDF_CFar** | ApplyCA_CFAR, ApplyOS_CFAR |
