# LiDAR API Reference

Summary of the main public LiDAR classes. For full details see `docs/API.md` in the repository.

---

## Core

| Class | Description |
|-------|-------------|
| **RDF_LidarSettings** | Scanner config: m_Range, m_RayCount, m_UpdateInterval, etc. |
| **RDF_LidarSample** | Single-ray result: m_Hit, m_Start, m_End, m_HitPos, m_Distance, m_Entity |
| **RDF_LidarScanner** | `Scan(IEntity, array)`, `SetSampleStrategy()`, `GetSettings()` |

---

## Visual

| Class | Description |
|-------|-------------|
| **RDF_LidarVisualSettings** | m_DrawPoints, m_DrawRays, m_PointSize, m_RayAlpha, m_RenderWorld, m_UseBatchedMesh |
| **RDF_LidarVisualizer** | `Render(subject, scanner)`, `GetLastSamples()` |
| **RDF_LidarColorStrategy** | BuildPointColor, BuildRayColorAtT |

---

## Strategies

| Class | Description |
|-------|-------------|
| **RDF_LidarSampleStrategy** | `BuildDirection(index, count)` |
| **RDF_UniformSampleStrategy** | Default uniform sphere |
| **RDF_HemisphereSampleStrategy** | Hemisphere |
| **RDF_ConicalSampleStrategy** | Conical sector |
| **RDF_SweepSampleStrategy** | Rotating sector (radar-style) |

---

## Demo

| Class | Description |
|-------|-------------|
| **RDF_LidarAutoRunner** | SetDemoEnabled, StartWithConfig, SetScanCompleteHandler |
| **RDF_LidarDemoConfig** | CreateDefault, CreateConical, CreateSweep, etc. |
| **RDF_LidarDemoCycler** | Cycle, StartAutoCycle, StopAutoCycle |
