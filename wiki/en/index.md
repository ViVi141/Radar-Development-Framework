# Radar Development Framework

Welcome to the **Radar Development Framework (RDF)** documentation.

RDF is a modular dual-sensor framework for **Arma Reforger**:

- **LiDAR** — Laser point-cloud scanning, visualization, demos
- **Electromagnetic Radar** — Full physics (FSPL, RCS, Doppler, SNR), PPI HUD

Both systems share a common sampling core and extension interfaces.
