# Project Overview

**Radar Development Framework (RDF)** is a lightweight, modular **dual-sensor framework** for **Arma Reforger** that implements:

- **LiDAR** — Laser radar ray point-cloud scanning, visualization, and demos
- **Electromagnetic Radar** — Full physics simulation (propagation attenuation, RCS, Doppler, SNR) with PPI scan-circle HUD

Both systems share a common sampling core and extension interfaces, yet run independently without interfering with each other.

> **⚠️ In development** — The radar module and demos are still under development. Do not use in production. The LiDAR module is relatively stable and can be used with caution.

---

## Features Overview

### LiDAR Module

- Configurable ray count, range, and interval; multiple sampling strategies (uniform, hemisphere, conical, stratified, scanline, radar sweep)
- Point cloud + gradient ray visualization; pluggable color strategies; point-cloud-only mode (solid background)
- CSV export, statistics/filtering, scan-complete callbacks; server-authoritative multiplayer network sync

### Electromagnetic Radar Module

- **Physics**: FSPL, atmospheric attenuation, rain attenuation, radar equation, SNR, system noise, Doppler shift
- **RCS models**: Analytic sphere/plate/cylinder RCS, material reflectivity lookup, entity bounding-box estimation, terrain diffuse reflection
- **Operating modes**: Pulse, CW (continuous wave), FMCW, Phased Array
- **Visualization**: 3D world pillar markers, ASCII console map, PPI circular scan HUD
- **HUD**: Built entirely via `WorkspaceWidget.CreateWidgetInWorkspace`; no `.layout` files required
- **Advanced**: SAR processor, ECM/jamming, automatic target classification
- **Demo system**: Six presets, auto-cycle, Bootstrap auto-start

```

  Lidar/
    Core/       Scan core: settings, types, scanner, sample strategies
    Visual/     Point cloud/ray rendering, color strategies
    Util/       Subject resolver, CSV export, stats/filter, callbacks
    Demo/       AutoRunner, DemoConfig, Cycler, Bootstrap
    Network/    Server-authoritative sync

  Radar/
    Core/       EMWaveParameters, RadarSample, RadarSettings, RadarScanner
    Physics/    Propagation, RCS model, radar equation, Doppler processor
    Modes/      Pulse / CW / FMCW / PhasedArray
    Visual/     Color strategies, PPI/A-Scope, world markers, text map
    Advanced/   SAR processor
    ECM/        Jamming model
    Classification/  Target classifier
    Demo/       RadarDemoConfig, AutoRunner, StatsHandler, Cycler, Bootstrap
    UI/         RDF_RadarHUD (CanvasWidget PPI)
    Tests/      Unit tests + benchmarks
    Util/       RadarExport

  EM/
    EMVoxelField.c, EMPassiveSensor.c
    Network/, Visual/, Tests/
```
