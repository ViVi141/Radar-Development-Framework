# Radar Quick Start

## Preset Configurations

| Factory Method | Band | Typical Use | Max Range |
|----------------|------|-------------|-----------|
| `CreateHelicopterRadar()` | X (10 GHz) | Helicopter forward sector | 8 000 m |
| `CreateXBandSearch()` | X (10 GHz) | Ground search/surveillance | 3 000 m |
| `CreateAutomotiveRadar()` | Ka (77 GHz) | Vehicle FMCW collision | 150 m |
| `CreateWeatherRadar()` | S (3 GHz) | Weather detection | 5 000 m |
| `CreatePhasedArrayRadar()` | C (5.6 GHz) | Multi-target phased array | 2 000 m |
| `CreateLBandSurveillance()` | L (1.3 GHz) | Long-range early warning | 5 000 m |

---

## Start Demo

```c
// Start with helicopter radar preset
RDF_RadarAutoRunner.StartWithConfig(RDF_RadarDemoConfig.CreateHelicopterRadar());

// Start with X-band search
RDF_RadarAutoRunner.StartWithConfig(RDF_RadarDemoConfig.CreateXBandSearch());

// Stop
RDF_RadarAutoRunner.Stop();
```

---

## Preset Cycler

```c
// Switch preset by index (0=X-band, 1=Automotive, 2=Weather, 3=PhasedArray, 4=L-band, 5=Helicopter)
RDF_RadarDemoCycler.StartIndex(2);

// Auto-cycle every 15 seconds
RDF_RadarDemoCycler.StartAutoCycle(15.0);
RDF_RadarDemoCycler.StopAutoCycle();

// Manually cycle to next preset
RDF_RadarDemoCycler.Cycle();
```

---

## HUD Control

```c
// Show / hide HUD
RDF_RadarHUD.Show();
RDF_RadarHUD.Hide();

// Update mode label
RDF_RadarHUD.SetMode("Heli Radar");

// Set PPI display range (should match scanner m_Range)
RDF_RadarHUD.SetDisplayRange(8000.0);
```

---

## Programmatic Single Scan

```c
void QuickDemo(IEntity radarEntity)
{
    RDF_RadarDemoConfig cfg = RDF_RadarDemoConfig.CreateXBandSearch(512);
    RDF_RadarScanner scanner = cfg.BuildRadarScanner();
    array<ref RDF_LidarSample> samples = new array<ref RDF_LidarSample>();
    scanner.Scan(radarEntity, samples);
    RDF_RadarExport.PrintScanSummary(samples);
}
```
