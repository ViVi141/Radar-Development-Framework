# Quick Start

Both LiDAR and Radar demos are **disabled by default**. You must enable them explicitly.

---

## Enabling Bootstrap (Auto-start)

Enable demos to run automatically when the game starts:

```c
// Enable radar demo
SCR_BaseGameMode.SetRadarBootstrapEnabled(true);

// Enable LiDAR demo (optional)
SCR_BaseGameMode.SetBootstrapEnabled(true);
```

Or edit the static variables in the source files:

```c
// RDF_RadarAutoBootstrap.c
protected static bool s_RadarBootstrapEnabled = true;

// RDF_LidarAutoBootstrap.c
protected static bool s_BootstrapEnabled = true;
```

---

## LiDAR Quick Start

```c
// Start with a preset config
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateDefault(256));
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateConical(25.0, 256));
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateSweep(30.0, 20.0, 45.0, 512));

// Auto-cycle strategies
RDF_LidarDemoCycler.StartAutoCycle(10.0);

// Enable / disable
RDF_LidarAutoRunner.SetDemoEnabled(true);
RDF_LidarAutoRunner.SetDemoEnabled(false);
```

See [1.2.1 LiDAR Quick Start](1.2.1-lidar-quick-start.md) for details.

---

## Radar Quick Start

```c
// Enable bootstrap
SCR_BaseGameMode.SetRadarBootstrapEnabled(true);

// Or start with a specific config
RDF_RadarAutoRunner.StartWithConfig(RDF_RadarDemoConfig.CreateHelicopterRadar());
RDF_RadarAutoRunner.StartWithConfig(RDF_RadarDemoConfig.CreateXBandSearch());

// Auto-cycle presets (every 15 seconds)
RDF_RadarDemoCycler.StartAutoCycle(15.0);

// Show / hide HUD
RDF_RadarHUD.Show();
RDF_RadarHUD.Hide();

// Update PPI display range (match scanner config)
RDF_RadarHUD.SetDisplayRange(2500.0);
```

See [1.2.2 Radar Quick Start](1.2.2-radar-quick-start.md) for details.
