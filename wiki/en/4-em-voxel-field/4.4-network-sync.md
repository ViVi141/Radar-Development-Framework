# EM Voxel Field Network Sync

## EMFieldNetworkAPI

Base interface for network integration. Default empty implementation.

---

## EMFieldNetworkComponent

Server-authoritative implementation:

- **Server**: Tick decay, broadcast detection results via unreliable RPC
- **Client**: Receive and store results; client → server RPC to register sectors
- `[RplProp]` sync for radar origin and state

---

## EMDetectionResult

Serializable detection result (comma-separated format) for RPC payload.
