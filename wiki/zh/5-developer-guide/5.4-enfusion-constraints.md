# Enfusion 脚本约束

| 约束条件 | 正确做法 |
|----------|----------|
| 不允许使用三元运算符 | 使用 `if/else` 替代 `? :` |
| 不允许使用函数式类型转换 | 使用 `(float)x` 替代 `float(x)` |
| 不允许使用 `string.ToLower()` | 直接与大小写字面量进行比较 |
| 不允许枚举转整数 | 预先填充 `EMyEnum[]` 数组用于查找 |
| `string.Format` 使用位置占位符 | 使用 `%1 %2` 替代 `%d %.1f` |
| `ToString()` 不能作用于临时表达式 | 先赋值给局部变量 |
| `out` 是关键字 | 使用其他名称，例如 `outSamples` |
| 非 ASCII 字符导致语法错误 | 保持文件仅使用 ASCII 字符 |
