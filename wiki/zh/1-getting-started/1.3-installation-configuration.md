# 安装与配置

## 安装

1. 将 **Radar Development Framework** 插件添加到您的 Arma Reforger Workbench 项目中。
2. 确保在您的场景或游戏模式中启用该插件。
3. 该框架为 **按需启用**：在通过 Bootstrap 或程序化 API 显式启用前，它不会执行任何操作。

---

## 插件配置

- **项目 ID**：`RadarDevelopmentFramework`
- **addon.gproj**：标准的 Arma Reforger 插件项目描述文件。
- **依赖项**：请参阅 `addon.gproj` 以获取所需的引擎模块。

---

## Bootstrap 配置

### LiDAR Bootstrap（`RDF_LidarAutoBootstrap.c`）

```c
SCR_BaseGameMode.SetBootstrapEnabled(true);           // 启用 LiDAR 演示（游戏启动时）
SCR_BaseGameMode.SetBootstrapAutoCycle(true);         // 启用策略自动循环
SCR_BaseGameMode.SetBootstrapAutoCycleInterval(10.0); // 循环间隔（单位：秒）
```

### Radar Bootstrap（`RDF_RadarAutoBootstrap.c`）

```c
SCR_BaseGameMode.SetRadarBootstrapEnabled(true);      // 启用雷达演示（游戏启动时）
SCR_BaseGameMode.SetRadarBootstrapAutoCycle(true);    // 启用预设自动循环
SCR_BaseGameMode.SetRadarBootstrapShowHUD(true);      // 显示 HUD（默认）
SCR_BaseGameMode.SetRadarBootstrapShowAScope(false);  // 显示 A-Scope（可选）
SCR_BaseGameMode.SetRadarBootstrapShowVoxel(false);   // 显示体素调试信息（可选）
```

---

## 通过源码配置

您也可以直接在 Bootstrap 文件中修改静态变量：

**RDF_LidarAutoBootstrap.c**：

- `s_BootstrapEnabled` — 启用 LiDAR 演示（默认：false）
- `s_BootstrapAutoCycle` — 启用自动策略切换（默认：false）
- `s_BootstrapAutoCycleInterval` — 循环间隔（单位：秒）

**RDF_RadarAutoBootstrap.c**：

- `s_RadarBootstrapEnabled` — 启用雷达演示（默认：false）
- `s_RadarBootstrapShowHUD` — 显示 HUD（默认：true）
- `s_RadarBootstrapConfigIdx` — 默认预设索引（0–5）
