# 项目概述

**雷达开发框架（RDF）** 是为 **Arma Reforger** 开发的轻量级、模块化的**双传感器框架**，实现了以下功能：

- **激光雷达（LiDAR）** — 激光雷达射线点云扫描、可视化与演示
- **电磁雷达** — 完整物理仿真（传播衰减、RCS、多普勒、信噪比），配备PPI扫描圆圈HUD

两个系统共享统一的采样核心与扩展接口，但彼此独立运行，互不干扰。

> **⚠️ 开发中** — 雷达模块与演示功能仍在开发中，**请勿用于生产环境**。激光雷达模块相对稳定，可谨慎使用。

---

## 功能概览

### 激光雷达模块

- 可配置射线数量、探测范围与采样间隔；支持多种采样策略（均匀、半球、锥形、分层、扫描线、雷达扫描）
- 点云 + 射线梯度可视化；可插拔的颜色策略；仅点云模式（纯色背景）
- CSV导出、统计/过滤、扫描完成回调；服务器权威的多人游戏网络同步

### 电磁雷达模块

- **物理模型**：FSPL、大气衰减、雨衰减、雷达方程、信噪比、系统噪声、多普勒频移
- **RCS模型**：解析球体/平板/圆柱体RCS、材料反射率查找表、实体包围盒估算、地形漫反射
- **工作模式**：脉冲、连续波（CW）、FMCW、相控阵
- **可视化**：3D世界柱状标记、ASCII控制台地图、PPI圆形扫描HUD
- **HUD**：完全通过 `WorkspaceWidget.CreateWidgetInWorkspace` 构建；无需 `.layout` 文件
- **高级功能**：SAR处理器、电子对抗/干扰、自动目标分类
- **演示系统**：六种预设、自动循环、Bootstrap自动启动

## 项目结构

```
scripts/Game/RDF/
  Lidar/
    Core/       采样核心：设置、类型、扫描器、采样策略
    Visual/     点云/射线渲染、颜色策略
    Util/       主体解析器、CSV导出、统计/过滤、回调
    Demo/       AutoRunner、DemoConfig、Cycler、Bootstrap
    Network/    服务器权威同步

  Radar/
    Core/       EMWaveParameters、RadarSample、RadarSettings、RadarScanner
    Physics/    传播、RCS模型、雷达方程、多普勒处理器
    Modes/      脉冲 / CW / FMCW / 相控阵
    Visual/     颜色策略、PPI/A-波形、世界标记、文本地图
    Advanced/   SAR处理器
    ECM/        干扰模型
    Classification/  目标分类器
    Demo/       RadarDemoConfig、AutoRunner、StatsHandler、Cycler、Bootstrap
    UI/         RDF_RadarHUD（CanvasWidget PPI）
    Tests/      单元测试 + 性能基准
    Util/       RadarExport

  EM/
    EMVoxelField.c, EMPassiveSensor.c
    Network/, Visual/, Tests/
```
