# LiDAR 快速入门

## 使用预设配置启动

```c
// 默认均匀采样（256 条射线）
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateDefault(256));

// 三色距离渐变（绿色→黄色→红色）
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateThreeColor(512));

// 半球采样
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateHemisphere(256));

// 锥形扇区（半角 25°，256 条射线）
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateConical(25.0, 256));

// 扫描策略（雷达式旋转扇区）
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateSweep(30.0, 20.0, 45.0, 512));
```

---

## 策略循环

```c
// 每 10 秒自动循环所有策略
RDF_LidarDemoCycler.StartAutoCycle(10.0);

// 停止自动循环
RDF_LidarDemoCycler.StopAutoCycle();

// 手动切换到下一个策略
RDF_LidarDemoCycler.Cycle(256);

// 从指定索引开始
RDF_LidarDemoCycler.StartIndex(2, 256);
```

---

## 启用 / 禁用演示

```c
RDF_LidarAutoRunner.SetDemoEnabled(true);
RDF_LidarAutoRunner.SetDemoEnabled(false);
```

---

## LiDAR HUD（可选）

```c
RDF_LidarAutoRunner.StartWithConfig(RDF_LidarDemoConfig.CreateDefault(512));
RDF_LidarHUD.Show();
RDF_LidarHUD.AttachToAutoRunner();  // HUD 在每次扫描后更新
```

---

## 仅逻辑扫描（无可视化）

```c
RDF_LidarScanner scanner = new RDF_LidarScanner();
scanner.GetSettings().m_Range = 80.0;
scanner.GetSettings().m_RayCount = 128;

array<ref RDF_LidarSample> samples = new array<ref RDF_LidarSample>();
IEntity subject = RDF_LidarSubjectResolver.ResolveLocalSubject(true);
scanner.Scan(subject, samples);

int hits = RDF_LidarSampleUtils.GetHitCount(samples);
RDF_LidarSample closest = RDF_LidarSampleUtils.GetClosestHit(samples);
```

参见 [LiDAR API 参考](../6-api-reference/6.1-lidar-api.md) 以获取完整 API。
