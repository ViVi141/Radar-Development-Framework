# 雷达物理处理流程

对于每个雷达探测到的射线，`RDF_RadarScanner.ApplyRadarPhysics()` 会按以下顺序执行这些步骤：

---

## 10步处理流程

| 步骤 | 描述 |
|------|------|
| 1. **FSPL** | 自由空间路径损耗：`20·log10(d) + 20·log10(f) + 92.45 dB` |
| 2. **大气衰减** | 依赖频率与距离（ITU-R P.676） |
| 3. **雨衰减** | 可选的雨衰减（ITU-R P.838） |
| 4. **RCS（雷达散射截面）** | 实体：包围盒 × 材质；地形：漫反射区域散射 |
| 5. **雷达方程** | `Pr = Pt·Gt·Gr·σ·λ² / (4π)³ / R⁴ / L` |
| 6. **距离门限** | 过滤近场盲区 |
| 7. **噪声** | `kTBF`，计算信噪比（SNR） |
| 8. **多普勒处理** | `fd = 2·vr·f / c`（径向速度） |
| 9. **检测** | `SNR ≥ m_DetectionThreshold` |
| 10. **杂波滤波** | 可选的MTI/MTD用于静止目标 |

---

## 核心组件

- **RDF_RadarPropagation** — 自由空间路径损耗、大气衰减、雨衰减
- **RDF_RCSModel** — 分析球体/板/柱体模型、材质查找表、实体估算、地形漫反射
- **RDF_RadarEquation** — 接收功率、噪声功率、信噪比
- **RDF_DopplerProcessor** — 径向速度、多普勒频移、MTI
- **RDF_CFar** — CA-CFAR / OS-CFAR（PoC）用于自适应检测门限
