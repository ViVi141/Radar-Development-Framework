# 雷达核心组件

## RDF_EMWaveParameters

电磁波参数：频率、波长、功率、脉冲重复频率、天线增益等。

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `m_CarrierFrequency` | float | 10e9 | 载波频率（Hz） |
| `m_Wavelength` | float | 0.03 | 波长（m） |
| `m_BandName` | string | "X" | 波段（L/S/C/X/Ku/K/Ka/V/W） |
| `m_TransmitPower` | float | 1000.0 | 发射功率（W） |
| `m_PulseWidth` | float | 1e-6 | 脉冲宽度（s） |
| `m_PRF` | float | 1000.0 | 脉冲重复频率（Hz） |
| `m_AntennaGain` | float | 30.0 | 天线增益（dBi） |
| `m_BeamwidthAzimuth` | float | 3.0 | 方位波束宽度（度） |
| `m_BeamwidthElevation` | float | 3.0 | 仰角波束宽度（度） |

**方法**：`CalculateWavelength()`，`DetermineBand()`，`Validate()`，`GetDescription()`

## RDF_RadarSample

继承自 `RDF_LidarSample`。新增：`m_ReceivedPower`，`m_SignalToNoiseRatio`，`m_RadarCrossSection`，`m_PathLoss`，`m_AtmosphericLoss`，`m_DopplerFrequency`，`m_TargetVelocity`，`m_TimeOfFlight` 等。

---

## RDF_RadarSettings

继承自 `RDF_LidarSettings`。新增：`m_EMWaveParams`，`m_RadarMode`，`m_DetectionThreshold`，`m_EnableDopplerProcessing`，`m_EnableMTI`，`m_EnableMTD`，大气/雨/ RCS/ CFAR 选项等。
