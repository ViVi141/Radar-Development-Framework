# 雷达高级功能

## SAR处理器（RDF_SARProcessor）

用于成像的合成孔径雷达处理器。

```c
void AddScan(array<ref RDF_LidarSample> samples, vector platformPos)
void GenerateSARImage(float areaSize, int resolution, vector areaCenter, ...)
void VisualiseImage(...)
void Reset()
int GetSnapshotCount()
```

`m_MaxSnapshots`（默认值为32）限制历史快照数量。

---

## ECM与干扰（RDF_JammingModel）

| 方法 | 描述 |
|------|------|
| `CalculateNoiseJammingJS(Pj, Gj, Gr, λ, Rj, Br)` | 计算噪声干扰的J/S比（dB） |
| `ApplyNoiseJamming(samples, jsDB, thresholdDB)` | 对样本应用噪声干扰 |
| `InjectFalseTargets(samples, count, x, y, z, ...)` | 注入欺骗性虚假目标 |
| `CalculateChaffRCS(numDipoles, wavelength)` | 计算箔条云的RCS（平方米） |

---

## 目标分类（RDF_TargetClassifier）

**ERadarTargetClass**: 未知、步兵、轻型车辆、重型车辆、飞机、静态物体、小型无人机、舰船

| 方法 | 描述 |
|------|------|
| `ClassifyTarget(sample)` | 根据RCS和速度规则对单个样本进行分类 |
| `ClassifyAll(samples, out outClasses)` | 批量分类 |
| `GetClassName(cls)` | 枚举值转为可读字符串 |
| `PrintStats(samples)` | 将分类统计信息打印到日志中 |
