# 雷达HUD与显示

## RDF_RadarHUD

完全由脚本驱动的雷达HUD。使用 `WorkspaceWidget.CreateWidgetInWorkspace` 构建；无需 `.layout` 文件。位置：左下角（在1920×1080分辨率下，X=20，Y=820）。

---

## 布局

```
+==========================================+
| [>] RADAR              X-Band Pulse     |  <- 标题 + 模式
+------------------------------------------+
|               N (North)                  |
|         CanvasWidget PPI 圆圈            |
|   W (West)   (+) 玩家   E (East)         |
|               S (South)                  |
|  内环 = 50% 距离，外环 = 100%            |
+------------------------------------------+
| SNR  65.2 dB   检测数  23 / 512 条射线     |
| RCS  +18.0 dBsm   速度  3.2 m/s          |
| 距离  30 m  ..  1447 m                  |
+------------------------------------------+
```

---

## 目标颜色编码

| 条件 | 颜色（ARGB） | 半径 |
|------|--------------|------|
| 实体，RCS > 10 dBsm | 亮绿色 #00FF44 | 4 px |
| 实体，RCS ≤ 10 dBsm | 青色 #00CCCC | 3 px |
| 地形，SNR > 20 dB | 黄色 #FFCC00 | 2 px |
| 弱地形 | 深橙色 #AA6600 | 2 px |

---

## 主要方法

```c
static RDF_RadarHUD GetInstance()
static void Show()
static void Hide()
static bool IsVisible()
static void SetMode(string modeName)
static void SetDisplayRange(float rangeM)
static void FeedSamples(array<ref RDF_LidarSample> samples)  // 无AutoRunner时使用
static void AttachToAutoRunner()
static void DetachFromAutoRunner()
```

**节流控制**：HUD更新频率最多每0.5秒一次，以防止闪烁。
