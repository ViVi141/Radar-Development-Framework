# 雷达演示系统

## RDF_RadarAutoRunner

单例演示驱动程序。管理扫描循环、回调和世界标记。

| 方法 | 描述 |
|------|------|
| `static GetInstance()` | 获取单例（懒加载初始化） |
| `static void StartWithConfig(RDF_RadarDemoConfig cfg)` | 使用预设配置启动 |
| `static void Stop()` | 停止并清理 |
| `static void SetScanCompleteHandler(RDF_LidarScanCompleteHandler handler)` | 注册回调函数 |
| `static GetLastSamples()` | 获取最后一次扫描的副本（防御性访问） |
| `static bool IsRunning()` | 演示是否正在运行 |

---

## RDF_RadarAutoBootstrap

`modded class SCR_BaseGameMode`。游戏启动时自动启动雷达演示。**默认禁用**。

```c
SCR_BaseGameMode.SetRadarBootstrapEnabled(true);
```
