# LiDAR 核心组件

## RDF_LidarSettings

扫描仪配置。运行时会被限制在安全范围内。

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `m_Enabled` | bool | true | 启用扫描 |
| `m_Range` | float | 50.0 | 扫描半径（米）。限制范围为 [0.1, 100000.0] |
| `m_RayCount` | int | 512 | 射线数量。最小值为 1 |
| `m_UpdateInterval` | float | 5.0 | 扫描间隔（秒）。最小值为 0.01 |
| `m_OriginOffset` | vector | "0 0 0" | 原点偏移（当启用 m_UseLocalOffset 时） |
| `m_TraceFlags` | int | WORLD \| ENTS | 追踪标志 |
| `m_LayerMask` | int | Projectile | 物理层掩码 |
| `m_UseBoundsCenter` | bool | true | 使用包围盒中心作为原点 |
| `m_UseLocalOffset` | bool | true | m_OriginOffset 在局部空间中的值 |

**方法**：
- `void Validate()` — 验证并限制参数（扫描前自动调用）。

## RDF_LidarSample

单条射线采样结构体。

| 字段 | 类型 | 描述 |
|------|------|------|
| `m_Index` | int | 射线索引 |
| `m_Hit` | bool | 是否命中 |
| `m_Start` | vector | 射线起点 |
| `m_End` | vector | 射线终点（最大距离） |
| `m_Dir` | vector | 单位方向向量 |
| `m_HitPos` | vector | 命中位置（或射线终点） |
| `m_Distance` | float | 命中距离（或最大范围） |
| `m_Entity` | IEntity | 命中的实体（如有） |
| `m_ColliderName` | string | 碰撞器名称 |
| `m_Surface` | GameMaterial | 命中表面材质 |

## RDF_LidarScanner

主扫描器类。

```c
RDF_LidarScanner(RDF_LidarSettings settings = null)
RDF_LidarSettings GetSettings()
void Scan(IEntity subject, array<ref RDF_LidarSample> outSamples)
void SetSampleStrategy(RDF_LidarSampleStrategy strategy)
RDF_LidarSampleStrategy GetSampleStrategy()
```

**默认采样策略**：`RDF_UniformSampleStrategy`。
