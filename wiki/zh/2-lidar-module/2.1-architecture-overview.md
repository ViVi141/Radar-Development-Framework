# LiDAR架构概览

LiDAR模块提供基于激光的点云扫描、可视化及演示功能，是雷达模块扩展的基础层。

---

## 设计原则

- **按需启用**：仅在明确启用时运行，不影响其他模组。
- **模块化**：扫描核心、物理模拟、可视化及演示逻辑相互隔离，可独立替换。
- **可扩展性**：提供稳定的扩展接口（采样策略、着色策略、扫描完成回调）。

---

## 数据流

```
RDF_LidarSettings
       |
       v
RDF_LidarScanner.Scan(entity)
  +-- SampleStrategy.BuildDirection() x N
  +-- Trace(ray) -> RDF_LidarSample
  +-- Returns array<ref RDF_LidarSample>
       |
       v
RDF_LidarVisualizer.RenderWithSamples()  (可选：3D点云/射线)
       |
       v
RDF_LidarScanCompleteHandler.OnScanComplete()  (可选回调)
```

---

## 模块布局

| 目录 | 作用 |
|------|------|
| **Core/** | 设置、类型、扫描器、采样策略 |
| **Visual/** | 点云/射线渲染、着色策略 |
| **Util/** | 主体解析器、CSV导出、统计/过滤、回调 |
| **Demo/** | 自动运行器、演示配置、循环器、引导器 |
| **Network/** | 服务端权威同步（Rpl） |
| **UI/** | RDF_LidarHUD（PPI点云视图） |

---

## 继承关系

`RDF_RadarScanner` 继承自 `RDF_LidarScanner`，复用射线追踪流程。雷达模块在每次命中时调用 `ApplyRadarPhysics()`，但LiDAR模块本身不依赖雷达。
