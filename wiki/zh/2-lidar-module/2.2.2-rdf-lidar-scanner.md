# RDF_LidarScanner

主激光雷达扫描仪类。执行射线追踪并填充 `RDF_LidarSample` 数组。

---

## 构造函数

```c
RDF_LidarScanner(RDF_LidarSettings settings = null)
```

创建具有默认或自定义设置的扫描仪。

---

## 方法

| 方法 | 描述 |
|------|------|
| `RDF_LidarSettings GetSettings()` | 返回设置引用 |
| `void Scan(IEntity subject, array<ref RDF_LidarSample> outSamples)` | 执行扫描；填充 outSamples。进入时调用 `Validate()` |
| `void SetSampleStrategy(RDF_LidarSampleStrategy strategy)` | 注入自定义采样策略 |
| `RDF_LidarSampleStrategy GetSampleStrategy()` | 返回当前策略 |

---

## 使用示例

```c
RDF_LidarScanner scanner = new RDF_LidarScanner();
scanner.GetSettings().m_Range = 100.0;
scanner.GetSettings().m_RayCount = 256;

array<ref RDF_LidarSample> samples = new array<ref RDF_LidarSample>();
IEntity subject = RDF_LidarSubjectResolver.ResolveLocalSubject(true);
scanner.Scan(subject, samples);
```

**默认策略**：`RDF_UniformSampleStrategy`。
