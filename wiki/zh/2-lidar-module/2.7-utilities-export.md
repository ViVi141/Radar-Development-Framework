# LiDAR 工具与导出

## RDF_LidarSubjectResolver

解析扫描主体（本地玩家或车辆）。

| 方法 | 描述 |
|------|------|
| `static IEntity ResolveLocalSubject(bool preferVehicle = true)` | 解析本地玩家；在车内时，若 preferVehicle 为 true，则返回车辆根节点 |
| `static IEntity ResolveSubject(IEntity player, bool preferVehicle = true)` | 为指定玩家解析主体 |
| `static vector ResolveOrigin(IEntity player = null, bool preferVehicle = true)` | 解析主体原点 |

---

## RDF_LidarExport

将扫描结果导出为 CSV 格式。

| 方法 | 描述 |
|------|------|
| `static string GetCSVHeader()` | CSV 标题行 |
| `static string SampleToCSVRow(RDF_LidarSample sample)` | 格式化单个样本 |
| `static void PrintCSVToConsole(array<ref RDF_LidarSample> samples)` | 将完整扫描结果打印到控制台 |
| `static bool ExportToFile(array<ref RDF_LidarSample> samples, string path)` | 写入文件（覆盖原有内容） |
| `static bool AppendToFile(array<ref RDF_LidarSample> samples, string path, bool writeHeaderIfNew = true)` | 追加写入文件 |

---

## RDF_LidarSampleUtils

统计与过滤功能。

| 方法 | 描述 |
|------|------|
| `static RDF_LidarSample GetClosestHit(array<ref RDF_LidarSample> samples)` | 获取最近命中点 |
| `static RDF_LidarSample GetFurthestHit(array<ref RDF_LidarSample> samples)` | 获取最远命中点 |
| `static int GetHitCount(array<ref RDF_LidarSample> samples)` | 获取命中点数量 |
| `static void GetHitsInRange(samples, minDist, maxDist, outSamples)` | 根据距离范围过滤 |
| `static float GetAverageDistance(samples, bool hitsOnly = true)` | 获取平均距离 |

---

## RDF_LidarScanCompleteHandler

回调基类。重写 `OnScanComplete(array<ref RDF_LidarSample> samples)` 方法以处理扫描结果。
