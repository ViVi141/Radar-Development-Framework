# RDF_LidarSettings

`RDF_LidarScanner` 的扫描器配置。值在运行时通过 `Validate()` 自动限制在安全范围内。

---

## 字段

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `m_Enabled` | bool | true | 启用扫描 |
| `m_Range` | float | 50.0 | 扫描半径（米）。限制范围为 [0.1, 100000.0] |
| `m_RayCount` | int | 512 | 射线数量。运行时最小值为 1 |
| `m_UpdateInterval` | float | 5.0 | 扫描间隔（秒）。最小值为 0.01 |
| `m_OriginOffset` | vector | "0 0 0" | 原点偏移；与 `m_UseLocalOffset` 一起使用 |
| `m_TraceFlags` | int | TraceFlags.WORLD \| TraceFlags.ENTS | 追踪标志 |
| `m_LayerMask` | int | EPhysicsLayerPresets.Projectile | 物理层掩码 |
| `m_UseBoundsCenter` | bool | true | 使用实体包围盒中心作为原点 |
| `m_UseLocalOffset` | bool | true | 将 `m_OriginOffset` 视为局部空间偏移 |

---

## 方法

| 方法 | 描述 |
|------|------|
| `void Validate()` | 执行防御性验证和限制。在调用 `Scan()` 前自动调用。 |
