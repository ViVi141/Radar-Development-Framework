# 采样策略示例

采样策略定义了射线方向的生成方式。实现 `RDF_LidarSampleStrategy::BuildDirection(int index, int count)` 并通过 `SetSampleStrategy()` 注入。

---

## 内置策略

| 策略 | 描述 |
|------|------|
| **RDF_UniformSampleStrategy** | 均匀球面（类似斐波那契分布） |
| **RDF_HemisphereSampleStrategy** | 半球面（例如向上方向） |
| **RDF_ConicalSampleStrategy** | 锥形扇区 |
| **RDF_StratifiedSampleStrategy** | 分层采样 |
| **RDF_ScanlineSampleStrategy** | 扫描线模式 |
| **RDF_SweepSampleStrategy** | 旋转扇区（雷达式） |

---

## RDF_SweepSampleStrategy

雷达式旋转扇区。

```c
RDF_SweepSampleStrategy(float halfAngleDeg = 30.0, float sweepWidthDeg = 20.0, float sweepSpeedDegPerSec = 45.0)
```

- **halfAngleDeg** — 锥形半角（度）
- **sweepWidthDeg** — 扇区宽度（度）
- **sweepSpeedDegPerSec** — 旋转速度（度/秒）

射线方向分布在扇区内，且扇区随世界时间旋转。
