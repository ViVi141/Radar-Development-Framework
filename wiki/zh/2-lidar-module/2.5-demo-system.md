# LiDAR 演示系统

## RDF_LidarAutoRunner

LiDAR 演示的单一入口点。通过 `SetDemoEnabled` 和 `SetDemoConfig` / `StartWithConfig` 进行控制。

---

## 主要方法

| 方法 | 描述 |
|------|------|
| `static void SetDemoEnabled(bool enabled)` | 启用 / 禁用演示 |
| `static bool IsDemoEnabled()` | 返回演示状态 |
| `static void StartWithConfig(RDF_LidarDemoConfig cfg)` | 应用配置并启动演示 |
| `static void SetDemoConfig(RDF_LidarDemoConfig cfg)` | 设置配置（不自动启动） |
| `static RDF_LidarDemoConfig GetDemoConfig()` | 获取当前配置 |
| `static void SetScanCompleteHandler(RDF_LidarScanCompleteHandler handler)` | 注册扫描完成回调 |
| `static void SetDemoSampleStrategy(RDF_LidarSampleStrategy strategy)` | 设置采样策略 |
| `static void SetDemoColorStrategy(RDF_LidarColorStrategy strategy)` | 设置颜色策略 |
| `static void SetDemoRenderWorld(bool renderWorld)` | 仅点云渲染 vs 世界+点云渲染 |

---

## RDF_LidarDemoConfig

配置对象。可通过工厂预设创建：

- `CreateDefault(int rayCount = 256)`
- `CreateThreeColor(int rayCount = 512)`
- `CreateHemisphere(int rayCount = 256)`
- `CreateConical(float halfAngleDeg, int rayCount)`
- `CreateSweep(float halfAngleDeg, float sweepWidthDeg, float sweepSpeedDegPerSec, int rayCount)`

---

## RDF_LidarDemoCycler

循环切换采样策略：

```c
static void Cycle(int rayCount = 256)
static void StartIndex(int index, int rayCount = 256)
static void StartAutoCycle(float interval = 10.0)
static void StopAutoCycle()
static bool IsAutoCycling()
```
