# RDF_LidarSample

单射线采样结果结构体。包含几何信息和命中信息。

---

## 字段

| 字段 | 类型 | 描述 |
|------|------|------|
| `m_Index` | int | 射线索引 |
| `m_Hit` | bool | 射线是否命中物体 |
| `m_Start` | vector | 射线起始位置 |
| `m_End` | vector | 射线终点（若未命中则为最大距离） |
| `m_Dir` | vector | 单位方向向量 |
| `m_HitPos` | vector | 命中位置（若未命中则为射线终点） |
| `m_Distance` | float | 命中距离（若未命中则为最大探测范围） |
| `m_Entity` | IEntity | 命中实体（如有） |
| `m_ColliderName` | string | 命中碰撞器名称 |
| `m_Surface` | GameMaterial | 命中表面材质（如有） |

---

## 继承关系

`RDF_RadarSample` 继承自 `RDF_LidarSample`，并增加了雷达专用字段（信噪比、雷达截面、多普勒等）。使用 `RDF_RadarScanner.Scan()` 时，将采样结果强制转换为 `RDF_RadarSample` 类型，即可访问雷达数据。
