# LiDAR可视化

## RDF_LidarVisualizer

渲染点云和可选的射线。使用 `RDF_LidarVisualSettings` 设置参数。

```c
RDF_LidarVisualizer(RDF_LidarVisualSettings settings = null)
void Render(IEntity subject, RDF_LidarScanner scanner)
ref array<ref RDF_LidarSample> GetLastSamples()  // 返回防御性副本
```

---

## RDF_LidarVisualSettings

| 字段 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `m_DrawPoints` | bool | true | 是否绘制命中点 |
| `m_DrawRays` | bool | true | 是否绘制射线 |
| `m_ShowHitsOnly` | bool | false | 仅绘制命中点 |
| `m_PointSize` | float | 0.08 | 点球体半径 |
| `m_RayAlpha` | float | 0.25 | 射线透明度 |
| `m_RaySegments` | int | 6 | 射线渐变段数 |
| `m_UseDistanceGradient` | bool | true | 使用基于距离的渐变着色 |
| `m_RenderWorld` | bool | true | true = 游戏世界 + 点云；false = 仅点云 |
| `m_UseBatchedMesh` | bool | false | 批量网格渲染（减少 Shape 调用次数） |

---

## 颜色策略

实现 `RDF_LidarColorStrategy` 并注入到可视化器中：

- `int BuildPointColor(float dist, bool hit, float lastRange, RDF_LidarVisualSettings settings)`
- `int BuildRayColorAtT(float t, bool hit, RDF_LidarVisualSettings settings)`

**内置实现**：
- `RDF_DefaultColorStrategy` — 距离渐变
- `RDF_IndexColorStrategy` — 基于索引的颜色
- `RDF_ThreeColorStrategy` — 近/中/远（绿色→黄色→红色）
