## 一、设计目标

将现有**射线追踪主动雷达**扩展为基于**3D 体素电磁场**的电子战仿真系统，实现：

- 主动雷达 → 向空间注入能量，从体素场读取回波
- 被动雷达 (RWR/ESM) → 直接采样所在位置的体素场
- 电子干扰 (Jamming) → 向体素注入噪声，污染雷达回波
- 多径效应、旁瓣、LPI 等高级物理

## 三、核心架构

### 3.1 稀疏体素场 (Sparse Voxel Field)

全图稠密 3D 数组不可行，采用稀疏存储：

- **数据结构**: `map<体素索引, EMVoxelCell>` 或分层网格
- **体素尺寸**: 近距 5 m / 远距 50 m（分级精度）
- **活跃区域**: 仅雷达/干扰机波束扫过的扇区分配体素
- **生命周期 (TTL)**: 能量无注入时快速衰减并释放内存

### 3.2 体素单元结构 (EMVoxelCell)

```c
class EMVoxelCell
{
    float   m_fTotalPower;       // 总能量 (W 或 dBm)
    int     m_iFrequencyMask;    // 频段掩码 (bitwise)
    vector  m_vMainDir;          // 主能量来源方向
    float   m_fNoiseFloor;       // 环境噪声基线

    // 脉冲特性（用于 RWR 识别）
    int     m_iLastPulseTimestamp;
    float   m_fPulseInterval;    // 推断 PRF

    // 可选：信号描述符
    int     m_iWaveformType;     // 0=Pulse, 1=CW, 2=FMCW, 3=LPI
}
```

### 3.3 体素索引与映射

- 世界坐标 → 体素索引：`(x,y,z) / cellSize` 取整
- 按扇区/块管理，便于批量激活与释放

## 四、主动雷达：能量注入

### 4.1 流程

1. **发射**：根据天线增益和波束宽度，发射一簇射线
2. **传播 (Ray Marching)**：沿射线步进
   - 计算当前点在体素场中的索引
   - 按 1/R² 衰减将能量写入该体素
3. **碰撞**：射线击中实体 → 按 RCS 计算反射能量 → 启动反向回波射线
4. **体素散射**：云层/森林体素可引入能量弥散

### 4.2 与现有 RDF_RadarScanner 的集成

- 保留 `TraceMove` 做几何命中
- 新增：在射线路径上的体素中写入能量和方向
- 回波路径同理，写入回波能量

---

## 五、被动雷达 (RWR/ESM)：空间采样

被动探测器不发射，只采样所在位置体素：

```
localPower = VoxelField.GetPowerAt(sensorPos)
if (localPower > sensitivityThreshold)
    incomingDir = VoxelField.GetDominantDirection(sensorPos)
    freq = VoxelField.GetFrequency(sensorPos)
    UpdateRWRDisplay(incomingDir, freq, localPower)
```

### 5.1 电子干扰 (Jamming)

- 干扰机向特定方向的体素大量注入噪声功率
- 真实雷达回波经过被污染体素时 SNR 降低
- 自然实现压制/欺骗干扰

## 六、进阶物理与拟真

### 6.1 信号身份特征化

| 维度 | 说明 |
|------|------|
| PRF | 脉冲重复频率 → 解距离模糊能力 |
| PW | 脉冲宽度 → 距离分辨率 |
| 频率跳变 | 频段掩码存储，RWR 识别抗干扰模式 |
| LPI | FMCW 等低功率宽频 → 普通 RWR 难以察觉 |

### 6.2 天线旁瓣与增益

- 主瓣 + 旁瓣：`G(θ) ∝ |sinc(θ)|`
- 旁瓣射线：主轴周围发射低功率射线
- 体素表现：雷达周围形成“能量光环”，支持旁瓣干扰

### 6.3 多径与两径模型

- 直射 + 地面反射 → 相位干涉
- 低空目标：引入 `4πht/λR` 衰减因子
- 实现地形遮蔽（Terrain Masking）的电磁层面效果

### 6.4 动态 RCS 与闪烁

- 角度相关 RCS 查表
- 瑞利分布噪声模拟回波起伏 (Scintillation)
- 可选：喷气调制 (JEM) 作为机型“指纹”

### 6.5 信号处理逻辑

| 算法 | 用途 |
|------|------|
| CFAR | 恒虚警率 → 箔条可提高噪声基线，淹没真实目标 |
| 多普勒盲速 | 切向飞行 (Notching) 可脱离跟踪 |

## 七、物理差异与拟真极限

体素场 + 射线簇已超越多数游戏雷达，但与真实电磁物理仍存在以下差异及缓解建议。

### 7.1 波动光学效应：衍射 (Diffraction)

| 差异 | 射线为直线，遇障产生完美阴影区；真实电磁波可经边缘衍射进入阴影 |
|------|----------------------------------------------------------------|
| 影响 | 山后/建筑后目标在现实中可能仍被低频雷达弱信号探测到 |
| 缓解 | 射线擦过地形/建筑边缘时，向阴影区体素注入少量能量（刀锋绕射近似） |

### 7.2 极化特性 (Polarization)

| 差异 | 射线模型无极化；真实波有 H/V/圆极化及极化失配损耗 |
|------|--------------------------------------------------|
| 影响 | 垂直极化干扰对水平极化雷达效果减弱；极化可滤雨雪杂波 |
| 缓解 | 体素增加极化枚举；回波功率乘极化失配因子；复杂度高，建议作为可选 |

### 7.3 大气折射与 4/3 地球效应

| 差异 | 射线绝对直线；真实大气密度随高度变化，波束向下弯曲 |
|------|----------------------------------------------------|
| 影响 | 折射使探测略超地平线；波导效应可超视距数百公里 |
| 缓解 | 长距离射线施加微小向下曲率；波导作为高级可选特性 |

### 7.4 距离与速度模糊性 (Ambiguity)

| 差异 | 脚本直接给出 R、V；真实雷达通过脉冲延迟和多普勒推算，存在不确定性 |
|------|------------------------------------------------------------------|
| 影响 | 距离折叠（高 PRF 下远目标被误判为近距）；盲速（多普勒为 PRF 整数倍时消失） |
| 缓解 | 在采样/输出层引入 PRF、盲速等规则；模拟老式或特定雷达的不确定性 |

### 7.5 目标闪烁与角噪声 (Glint & Scintillation)

| 差异 | 模型多为点/静态 RCS；真实目标由多反射点干涉，回波起伏剧烈 |
|------|------------------------------------------------------------|
| 影响 | 追踪雷达波束中心抖动；防空导弹末端轨迹不稳 |
| 缓解 | 对回波功率加瑞利分布噪声；角噪声体现为波束中心随机偏移 |

### 7.6 接收机噪声基底 (Thermal Noise & KTB)

| 差异 | 常见逻辑为 `if (power > threshold)`；真实接收机存在随机热噪声 |
|------|----------------------------------------------------------------|
| 影响 | 远距目标在屏幕上应时隐时现；强干扰下 SNR 不足导致漏检 |
| 缓解 | 噪声基底 + 随机抖动；已有 kTBF，扩展为时变噪声即可 |

### 7.7 大气吸收与天气衰减 (Attenuation)

| 差异 | 通常只算 1/R⁴ 自由空间损耗 |
|------|----------------------------|
| 影响 | 毫米波在雨天性能下降；60 GHz 等频段大气吸收强，仅能近距使用 |
| 缓解 | 根据天气（晴/雨/雾）在传播路径上增加 `exp(-α·R)` 衰减；复用现有大气/雨衰模型 |

### 7.8 实施优先级

| 优先级 | 项目 | 难度 | 收益 |
|--------|------|------|------|
| 高 | 热噪声基底 | 低 | 显著 |
| 高 | 大气/天气衰减 | 低 | 显著 |
| 中 | 衍射近似 | 中 | 中等 |
| 中 | 目标闪烁噪声 | 中 | 中等 |
| 中 | 距离/速度模糊 | 中 | 中等 |
| 低 | 极化 | 高 | 可选 |
| 低 | 折射/波导 | 高 | 可选 |

## 八、无波引擎的补偿方案

### 8.1 时间片延迟传播

- 射线擦过障碍边缘 → 在边缘体素生成子射线进入阴影区（近似绕射）
- 地面反射 → 反射射线在另一体素累加能量（多径）

### 8.2 多径效应

- 射线击中地面后反射
- 被动雷达在特定高度检测到直射+反射双峰

## 九、性能与存储

### 9.1 存储估算（20 km × 20 km × 1 km）

| 分辨率 | 体素数 | 稀疏 1% | 单格 32B | 稀疏后 |
|--------|--------|---------|----------|--------|
| 50 m | 160M | 1.6M | — | ~50 MB |
| 10 m | 4G | 40M | — | ~1.3 GB |

**结论**：稀疏 + 分级 + 仅活跃扇区，目标 < 100 MB 内存

### 9.2 优化策略

| 策略 | 说明 |
|------|------|
| 扇区活跃化 | 仅被扫描扇区参与计算 |
| TTL 衰减 | 无注入则快速清零并释放 |
| 对象池 | 复用 EMVoxelCell，避免频繁 new |
| 服务端计算 | 客户端不维护体素场 |

### 9.3 网络同步

- 服务端：完整体素场 + 雷达/干扰机逻辑
- 客户端：仅接收探测结果（目标列表、RWR 告警、PPI 显示数据）
- 不同步体素场本身
